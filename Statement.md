## Problem Statement
Buying, selling, and investing in real estate often depends on accurate property valuation. In many cases, people rely on manual estimates, local opinions, or outdated listings, which can lead to inconsistent and biased price decisions. This project solves that challenge by building a machine learning based system that predicts house prices using key property attributes such as location, area, BHK, furnishing status, and other related factors.

## Scope of Project
The scope of this project includes:
- Collecting and using structured housing data for model training.
- Preprocessing both numerical and categorical features.
- Training a regression based prediction model.
- Saving and loading the trained model for reuse.

Out of scope:
- This is a CLI based project no UI is used
- The prices and estimates are not live so the data may be old, current data is of 2025
- Legal or financial advisory is not in this project 

## Target Users
- Property Dealers or sellers who want to know the price of their property
- Buyers who want to purchase a house for investment or personal use
- Students who want to see how basic ML works 

## High-Level Features
- Data driven prediction with the help of the standard example property and thier prices 
- prediction includes both the categorial(e.g. Balcony,location, facing etc ) and numerical features (BHK, squre feet area etc)
- Basic error handling of the input from the users for safety of the system resources 

## Summary
This project uses Linear Regression to train a machine learning model that predicts property prices, enabling sellers and buyers to determine fair property rates in their hometown
